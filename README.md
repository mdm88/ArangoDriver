# ArangoDriver

ArangoDriver is a C# driver for [ArangoDB](https://www.arangodb.com/) v3.5 NoSQL multi-model database. Driver implements and communicates with database backend through its [HTTP API](https://docs.arangodb.com/3.0/HTTP/index.html) interface for .Net Core.

**Based on [ArangoDB-NET](https://github.com/yojimbo87/ArangoDB-NET)**

## Installation

There are following ways to install the driver:

- ~~download and install nuget package~~
- clone ArangoDriver [repository](https://github.com/mdm88/ArangoDriver) and build [master branch](https://github.com/mdm88/ArangoDriver/tree/master) to have latest version
